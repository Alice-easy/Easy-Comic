# Easy Comic

一个使用 Flutter 构建的简单的漫画阅读器。

## ✨ 功能

- 浏览和阅读漫画
- ... (在这里添加更多的功能)

## 🚀 开始使用

按照以下步骤在本地运行此项目。

### 环境要求

- Flutter SDK: `^3.0.0`
- Dart SDK: `^3.0.0`

### 安装

1. 克隆仓库
   ```sh
   git clone https://github.com/your_username/easy_comic.git
   ```
2. 安装依赖
   ```sh
   flutter pub get
   ```
3. 运行应用
   ```sh
   flutter run
   ```
